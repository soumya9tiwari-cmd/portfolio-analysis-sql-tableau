# Investment Portfolio Analysis using SQL & Tableau

## Overview

This project presents an end-to-end investment portfolio analysis completed in two stages. The first stage focuses on extracting, cleaning, and analyzing portfolio data using SQL, while the second stage transforms the results into interactive Tableau dashboards for portfolio performance, risk assessment, and asset allocation analysis.

The objective was to evaluate portfolio composition, identify concentration risk, compare investment performance across asset classes, and provide data-driven recommendations to improve diversification and long-term portfolio stability.

---

## Project Structure

### Part 1 – SQL Portfolio Analysis

The SQL phase focused on preparing and analyzing the investment portfolio dataset.

Key tasks included:

- Data extraction and cleaning
- Multi-table joins
- Creation of SQL views
- Portfolio return calculations
- Rolling 12-, 24-, and 36-month return analysis
- Asset class performance evaluation
- Portfolio-level financial insights

**Skills Used**

- SQL
- Joins

- Window Functions
- Financial Data Analysis

### Part 2 – Tableau Dashboard & Visualization

The SQL outputs were used to build interactive Tableau dashboards that communicate portfolio insights visually.

The dashboard analyzes:

- Portfolio composition by asset class
- Portfolio concentration (Pareto Analysis)
- ETF vs. Stock performance
- Performance trends (2019–2022)
- Risk trends by asset class
- Risk vs. Return analysis
- KPI summary
- Portfolio diversification recommendations

## Key Insights

- Equity holdings represented the largest share of the portfolio, resulting in higher concentration risk.
- ETFs demonstrated more consistent performance than individual stocks across several asset classes.
- Risk and return showed a positive relationship, with equity investments exhibiting the highest volatility.
- Increasing exposure to fixed income and other non-equity assets could improve portfolio diversification and reduce overall portfolio risk. :contentReference[oaicite:1]{index=1}
